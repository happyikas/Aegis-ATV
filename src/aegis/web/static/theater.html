What's an ATV? Every action an AI agent attempts is captured as a fixed 2,080-dimensional vector — the Agent Trace Vector. Each band of the vector encodes a different facet of the action (who, what, how risky, how expensive). The Action Firewall reads specific bands to decide ALLOW / REQUIRE_APPROVAL / BLOCK. Watch the agent on the left; watch the vector on the right; watch which band lights up red when the firewall fires.
